# 题目08：奇怪的数列

## 题目类型
复杂逻辑处理

## 难度
中等偏难

## 知识点
数学规律、公式推导、数列计算

## 题目描述

有一个奇怪的数列，定义如下：
- 第1项：1² = 1
- 第2项：1² = 1
- 第3项：2² = 4
- 第4项：2² = 4
- 第5项：3² = 9
- 第6项：3² = 9
- ...

规律：第n项的值是 k²，其中 k = ⌈n/2⌉（向上取整，即(n+1)/2的整数部分）。

给定n，计算第n项的值。

## 输入格式

输入一个正整数n（1 <= n <= 10^9），表示要计算的项数。

## 输出格式

输出一个整数，表示第n项的值。

## 样例输入1

```
1
```

## 样例输出1

```
1
```

## 样例输入2

```
3
```

## 样例输出2

```
4
```

## 样例输入3

```
5
```

## 样例输出3

```
9
```

## 样例输入4

```
100
```

## 样例输出4

```
2500
```

**解释**：k = (100+1)/2 = 50，所以第100项 = 50² = 2500

## 提示

1. **算法思路**：
   - 根据规律，第n项的值是 k²，其中 k = ⌈n/2⌉
   - 在C++中，⌈n/2⌉ = (n+1)/2（整数除法）

2. **实现步骤**：
   - 计算 k = (n + 1) / 2
   - 计算 k²

3. **注意事项**：
   - 注意使用`long long`类型，因为n和k²可能很大
   - 注意整数除法的结果

## 要求

1. 注意处理大数（使用long long）
2. 代码简洁，逻辑清晰
