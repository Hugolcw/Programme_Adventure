# 题目04：简单计算器

## 题目类型
模拟器/状态机

## 难度
较难

## 知识点
栈的应用、表达式求值、运算符优先级、状态管理

## 题目描述

实现一个简单的计算器，能够计算只包含加法和乘法的表达式。表达式由数字、加号(+)和乘号(*)组成，没有括号。

例如：
- `2+3*4` = 2 + 12 = 14
- `1+2+3*4` = 1 + 2 + 12 = 15
- `2*3+4` = 6 + 4 = 10

**注意**：乘法优先级高于加法，需要先计算乘法。

## 输入格式

输入一行，包含一个表达式字符串，长度不超过1000。表达式只包含：
- 数字（0-9，可能有多位）
- 加号(+)
- 乘号(*)
- 空格（需要忽略）

## 输出格式

输出一个整数，表示表达式的计算结果。

## 样例输入1

```
2+3*4
```

## 样例输出1

```
14
```

## 样例输入2

```
1+2+3*4
```

## 样例输出2

```
15
```

## 样例输入3

```
2*3+4*5
```

## 样例输出3

```
26
```

## 样例输入4

```
1 + 2 * 3 + 4
```

## 样例输出4

```
11
```

## 提示

1. **算法思路**：
   - 使用栈来处理运算符优先级
   - 遇到数字，读取完整数字
   - 遇到运算符，根据优先级决定是否先计算
   - 乘法优先级高于加法，遇到乘法时立即计算

2. **实现步骤**：
   - 方法1：使用两个栈（数字栈和运算符栈）
   - 方法2：先处理所有乘法，再处理加法（推荐，更简单）

3. **方法2详细步骤**：
   - 遍历表达式，提取所有数字和运算符
   - 先处理所有乘法：遇到`*`时，将前一个数字和后一个数字相乘
   - 再处理所有加法：将所有剩余数字相加

4. **数字提取**：
   - 连续的数字字符组成一个数字
   - 使用循环读取完整的数字

## 要求

1. 不能使用STL容器（如stack、vector），需要用数组模拟栈
2. 注意处理空格
3. 注意数字可能是多位数
4. 代码逻辑清晰，注释适当

---

## 考查重点

- **栈的应用**：理解栈在表达式求值中的作用
- **运算符优先级**：如何处理不同优先级的运算符
- **状态管理**：如何管理解析过程中的状态
