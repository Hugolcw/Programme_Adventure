# 题目05：最大公约数（递归）

## 题目类型
递推递归

## 难度
中等

## 知识点
递归、欧几里得算法、递推关系

## 题目描述

使用递归的欧几里得算法计算两个正整数的最大公约数（GCD）。

欧几里得算法的递归公式：
- gcd(a, 0) = a
- gcd(a, b) = gcd(b, a % b) （当b ≠ 0时）

## 输入格式

输入若干对正整数a和b（1 <= a, b <= 10^9），每对占一行。输入以文件尾结束。

## 输出格式

对于每对输入，输出一个整数，表示a和b的最大公约数。

## 样例输入

```
48 18
100 25
17 13
```

## 样例输出

```
6
25
1
```

## 提示

1. **欧几里得算法**：
   - 基于原理：gcd(a, b) = gcd(b, a % b)
   - 当b=0时，a就是最大公约数

2. **递归三要素**：
   - **终止条件**：b == 0，返回a
   - **递归关系**：gcd(a, b) = gcd(b, a % b)
   - **递归调用**：调用自身计算gcd(b, a % b)

3. **函数定义**：
   ```cpp
   int gcd(int a, int b) {
       // 终止条件
       if (b == 0) return a;
       
       // 递归关系
       return gcd(b, a % b);
   }
   ```

4. **示例**：
   - gcd(48, 18) = gcd(18, 48 % 18) = gcd(18, 12)
   - gcd(18, 12) = gcd(12, 18 % 12) = gcd(12, 6)
   - gcd(12, 6) = gcd(6, 12 % 6) = gcd(6, 0)
   - gcd(6, 0) = 6

## 要求

1. 必须使用递归实现
2. 注意处理边界情况（a或b为0的情况）
3. 代码逻辑清晰，注释适当
