# 题目01：斐波那契数列

## 题目类型
基础递归

## 难度
简单

## 知识点
递归、递推关系、递归终止条件

## 题目描述

斐波那契数列是一个经典的递归问题。数列定义如下：
- F(0) = 0
- F(1) = 1
- F(n) = F(n-1) + F(n-2) （n ≥ 2）

请使用递归函数计算斐波那契数列的第n项。

## 输入格式

输入一个正整数n（0 <= n <= 30），表示要计算的项数。

## 输出格式

输出一个整数，表示斐波那契数列的第n项。

## 样例输入1

```
0
```

## 样例输出1

```
0
```

## 样例输入2

```
1
```

## 样例输出2

```
1
```

## 样例输入3

```
5
```

## 样例输出3

```
5
```

**解释**：F(5) = F(4) + F(3) = 3 + 2 = 5

## 样例输入4

```
10
```

## 样例输出4

```
55
```

## 提示

1. **递归三要素**：
   - **终止条件**：n == 0 或 n == 1
   - **递归关系**：F(n) = F(n-1) + F(n-2)
   - **递归调用**：调用自身计算F(n-1)和F(n-2)

2. **函数定义**：
   ```cpp
   long long fib(int n) {
       // 终止条件
       if (n == 0) return 0;
       if (n == 1) return 1;
       
       // 递归关系
       return fib(n-1) + fib(n-2);
   }
   ```

3. **注意事项**：
   - 注意n的范围，n最大为30
   - 可以使用`long long`类型避免溢出
   - 虽然递归实现简单，但效率较低（会重复计算）

## 要求

1. 必须使用递归实现
2. 注意处理边界情况（n=0和n=1）
3. 代码逻辑清晰，注释适当

- **双递归调用**：理解递归函数可以调用自身两次
- **递归思维**：将问题分解为相同类型的子问题
