# 题目06：回文数判断（递归）

## 题目类型
递推递归

## 难度
中等

## 知识点
递归、字符串递归、双指针递归思想

## 题目描述

判断一个正整数是否为回文数。回文数是指正读和反读都相同的数字。

例如：121是回文数，123不是回文数。

请使用递归方法实现。

## 输入格式

输入若干正整数n（1 <= n <= 10^9），每行一个。输入以文件尾结束。

## 输出格式

对于每个输入，如果是回文数，输出"yes"；否则输出"no"。

## 样例输入

```
121
123
1
```

## 样例输出

```
yes
no
yes
```

## 提示

1. **递归思路**：
   - 将数字转换为字符串，递归比较首尾字符
   - 或者递归提取数字的各位，构造反转数字

2. **方法1：字符串递归**（推荐）
   - 递归比较字符串的首尾字符
   - 如果首尾相同，递归判断中间部分
   - 如果首尾不同，返回false

3. **方法2：数字递归**
   - 递归提取数字的各位数字
   - 构造反转数字
   - 比较原数和反转数

4. **递归三要素**（方法1）：
   - **终止条件**：
     - 字符串长度为0或1：返回true（空串或单个字符都是回文）
   - **递归关系**：
     - 如果首尾字符相同：递归判断中间部分
     - 如果首尾字符不同：返回false
   - **递归调用**：调用自身判断子字符串

## 要求

1. 必须使用递归实现
2. 注意处理边界情况（单个数字）
3. 代码逻辑清晰，注释适当
