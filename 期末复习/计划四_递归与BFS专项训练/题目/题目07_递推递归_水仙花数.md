# 题目07：水仙花数（递归）

## 题目类型
递推递归

## 难度
中等

## 知识点
递归、数字递归分解、递归计算

## 题目描述

判断一个三位数是否为水仙花数。水仙花数是指一个n位数（n≥3），其各位数字的n次幂之和等于它本身。

对于三位数：abc是水仙花数当且仅当 a³ + b³ + c³ = abc

例如：153 = 1³ + 5³ + 3³ = 1 + 125 + 27 = 153，所以153是水仙花数。

请使用递归方法提取数字的各位并计算各位的立方和。

## 输入格式

输入一个三位正整数m（100 <= m <= 999）。

## 输出格式

如果m是水仙花数，输出"m is a narcissistic number."；否则输出"m is not a narcissistic number."。

## 样例输入1

```
153
```

## 样例输出1

```
153 is a narcissistic number.
```

## 样例输入2

```
123
```

## 样例输出2

```
123 is not a narcissistic number.
```

## 提示

1. **递归思路**：
   - 递归提取数字的各位数字
   - 递归计算各位数字的立方和

2. **递归三要素**：
   - **终止条件**：n == 0，返回0
   - **递归关系**：sum = (n % 10)³ + sumOfCubes(n / 10)
   - **递归调用**：调用自身处理去掉个位后的数字
