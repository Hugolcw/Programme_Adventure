# 答案03：数字序列

## 完整代码

见 `答案03_循环逻辑_数字序列.cpp`

## 解题思路

### 核心算法
使用递推思想，从已知的前两项开始，逐步计算后续项。

### 算法步骤
1. 特殊情况：n=1返回1，n=2返回2
2. 初始化：`a = 1`（第1项），`b = 2`（第2项）
3. 循环从第3项开始到第n项：
   - 计算下一项：`c = a + b`
   - 更新：`a = b`, `b = c`
4. 输出b（即第n项）

### 示例演示
```
n = 7
初始：a=1, b=2
j=3: c=1+2=3, a=2, b=3
j=4: c=2+3=5, a=3, b=5
j=5: c=3+5=8, a=5, b=8
j=6: c=5+8=13, a=8, b=13
j=7: c=8+13=21, a=13, b=21
结果：21
```

### 关键点
- **递推关系**：f(n) = f(n-1) + f(n-2)
- **边界处理**：前两项需要特殊处理
- **变量更新**：使用两个变量滚动计算，节省空间

## 测试用例

### 序列验证
- 第1项：1 ✓
- 第2项：2 ✓
- 第3项：1+2=3 ✓
- 第4项：2+3=5
- 第5项：3+5=8
- 第6项：5+8=13
- 第7项：8+13=21 ✓

## 常见错误

1. **边界处理错误**：忘记处理n=1和n=2的情况
2. **变量更新错误**：更新顺序应该是先a后b
3. **循环起始错误**：应该从j=3开始
