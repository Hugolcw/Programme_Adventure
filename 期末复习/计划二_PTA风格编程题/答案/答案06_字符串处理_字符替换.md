# 答案06：字符替换

## 完整代码

见 `答案06_字符串处理_字符替换.cpp`

## 解题思路

### 核心算法
遍历字符串的每个字符，如果等于oldChar，则替换为newChar。

### 算法步骤
1. 使用getline读取字符串（可能包含空格）
2. 读取oldChar和newChar
3. 遍历字符串的每个字符：
   - 如果 `s[i] == oldChar`，则 `s[i] = newChar`
4. 输出替换后的字符串

### 示例演示
```
s = "hello world"
oldChar = 'l', newChar = 'x'

i=0: s[0]='h' ≠ 'l', 不变
i=1: s[1]='e' ≠ 'l', 不变
i=2: s[2]='l' == 'l', 替换为'x' → s[2]='x'
i=3: s[3]='l' == 'l', 替换为'x' → s[3]='x'
i=4: s[4]='o' ≠ 'l', 不变
...
结果："hexxo worxd"
```

### 关键点
- **getline读取**：字符串可能包含空格，必须使用getline
- **字符比较**：使用 `==` 比较字符
- **直接修改**：string支持直接修改字符 `s[i] = newChar`

## 测试用例

### 测试用例1
**输入**：
```

l x
```
**输出**：
```
hexxo worxd
```
**说明**：所有'l'替换为'x'

### 测试用例2
**输入**：
```
programming
g G
```
**输出**：
```
proGramminG
```
**说明**：所有'g'替换为'G'

## 常见错误

1. **输入方式错误**：使用`cin >> s`会只读取第一个单词，应该用getline
2. **字符比较错误**：应该用单引号`'l'`而不是双引号`"l"`
3. **字符串长度**：使用`s.length()`获取长度
