# 答案01：数字反转

## 完整代码

见 `答案01_基础语法_数字反转.cpp`

## 解题思路

### 核心算法
使用循环提取数字的各位，同时构建反转后的数字。

### 算法步骤
1. 初始化 `reversed = 0`
2. 循环提取n的个位数字：`n % 10`
3. 将个位数字添加到反转数字：`reversed = reversed * 10 + n % 10`
4. 去掉n的个位：`n /= 10`
5. 重复直到n为0

### 示例演示
```
n = 123
第1次：reversed = 0*10 + 3 = 3,    n = 12
第2次：reversed = 3*10 + 2 = 32,   n = 1
第3次：reversed = 32*10 + 1 = 321, n = 0
结果：321
```

### 关键点
- **自动处理前导零**：100反转时，第一次得到0，但0*10+0还是0，直到遇到非零位才开始构建数字
- **循环终止条件**：`n > 0`，当n变为0时停止

## 测试用例

### 输入
```

```

### 输出
```

1

```

## 常见错误

1. **忘记初始化reversed**：应该初始化为0
2. **循环条件错误**：应该是`n > 0`而不是`n >= 0`
3. **输出格式错误**：每个测试用例输出后要换行
