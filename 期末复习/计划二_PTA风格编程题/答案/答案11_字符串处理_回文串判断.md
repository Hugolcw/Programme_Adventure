# 答案11：回文串判断

## 完整代码

见 `答案11_字符串处理_回文串判断.cpp`

## 解题思路

### 核心算法
使用双指针从两端向中间比较，忽略非字母数字字符，忽略大小写。

### 算法步骤
1. 初始化：`left = 0`, `right = s.length() - 1`
2. 循环比较（`left < right`）：
   - 跳过left指向的非字母数字字符
   - 跳过right指向的非字母数字字符
   - 转换为小写后比较
   - 如果不同，返回false
   - left++，right--
3. 循环结束，返回true

### 示例演示
```
s = "level"
left=0('l'), right=4('l'): tolower('l')==tolower('l') ✓
left=1('e'), right=3('e'): tolower('e')==tolower('e') ✓
left=2('v'), right=2('v'): left>=right，结束
结果：true

s = "A man a plan a canal Panama"
忽略空格和大小写后：amanaplanacanalpanama
双指针比较，是回文串
```

### 关键点
- **双指针**：从两端向中间移动
- **字符过滤**：使用isalnum判断是否为字母或数字
- **大小写忽略**：使用tolower转换为小写
- **cin.ignore()**：读取整行前需要忽略换行符

## 测试用例


### 验证
- "level"：正读和反读相同 ✓
- "Hello"：正读"Hello"，反读"olleH"，不同 ✗
- "A man a plan a canal Panama"：忽略空格和大小写后是回文 ✓

## 常见错误

1. **忘记cin.ignore()**：getline前需要忽略换行符
2. **字符过滤错误**：忘记跳过非字母数字字符
3. **大小写处理错误**：忘记转换为统一大小写
4. **边界条件**：left和right的移动顺序错误
