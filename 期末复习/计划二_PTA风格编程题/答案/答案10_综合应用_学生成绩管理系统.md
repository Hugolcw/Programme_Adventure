见 `答案10_综合应用_学生成绩管理系统.cpp`

## 解题思路

### 核心设计
使用结构体存储学生信息，实现多个功能：统计、查找、排序。

### 算法步骤
1. **输入和计算**：
   - 读取学生信息
   - 计算总分和平均分
   - 统计分数段人数

2. **查找最高分**：
   - 遍历数组找总分最大的学生

3. **排序**：
   - 使用冒泡排序按总分降序排列

4. **输出**：
   - 输出最高分学生
   - 输出分数段统计
   - 输出排序后的所有学生

### 关键点
- **结构体使用**：Student结构体存储学生信息
- **分数段判断**：getGrade函数根据平均分返回等级索引
- **冒泡排序**：自己实现排序算法
- **输出格式**：注意小数位数和格式

## 测试用例

### 输入
```
3
张三 85 90 88
李四 92 88 95
王五 78 82 80
```

### 输出
```
最高分: 李四 275 91.7
A: 1 B: 2 C: 0 D: 0 F: 0
李四 275 91.7
张三 263 87.7
王五 240 80.0
```

### 验证
- 张三：总分=263，平均分=87.7（B等级）✓
- 李四：总分=275，平均分=91.7（A等级）✓
- 王五：总分=240，平均分=80.0（B等级）✓
- 最高分：李四 ✓
- 分数段：A=1, B=2 ✓

## 常见错误

1. **平均分计算错误**：应该除以3.0而不是3（避免整数除法）
2. **排序错误**：冒泡排序的循环条件或交换逻辑错误
3. **分数段判断错误**：边界条件处理不当（如90分属于A还是B）
4. **输出格式错误**：小数位数或格式字符串错误
