# 答案12：数组最大值交换

## 完整代码

见 `答案12_指针引用_数组最大值交换.cpp`

## 解题思路

### 核心算法
使用指针遍历数组，找到最大值的位置，然后与第一个元素交换。

### 算法步骤
1. 初始化：`maxIndex = 0`（假设第一个元素最大）
2. 遍历数组（i从1到n-1）：
   - 如果 `arr[i] > arr[maxIndex]`，更新 `maxIndex = i`
3. 如果 `maxIndex != 0`，交换 `arr[0]` 和 `arr[maxIndex]`

### 示例演示
```
arr = [3, 1, 4, 1, 5], n = 5
初始：maxIndex = 0 (arr[0]=3)
i=1: arr[1]=1 < arr[0]=3, 不更新
i=2: arr[2]=4 > arr[0]=3, maxIndex=2
i=3: arr[3]=1 < arr[2]=4, 不更新
i=4: arr[4]=5 > arr[2]=4, maxIndex=4
交换：arr[0]和arr[4]交换
结果：[5, 1, 4, 1, 3]
```

### 关键点
- **指针参数**：函数接收指针，可以直接修改数组
- **数组访问**：使用`arr[i]`访问元素（指针可以像数组一样使用）
- **边界检查**：如果最大值就是第一个元素，不需要交换

## 常见错误

1. **指针使用错误**：忘记指针可以像数组一样使用
2. **交换逻辑错误**：交换时使用错误的索引
3. **边界检查**：忘记检查maxIndex是否为0
4. **数组访问**：使用`*arr`而不是`arr[i]`（虽然都可以，但后者更清晰）
