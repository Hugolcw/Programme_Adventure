# 答案05：矩阵转置

## 完整代码

见 `答案05_数组操作_矩阵转置.cpp`

## 解题思路

### 核心算法
矩阵转置：原矩阵a[i][j]对应转置矩阵b[j][i]。可以直接输出，不需要存储转置矩阵。

### 算法步骤
1. 读取矩阵大小n×m
2. 读取n×m的矩阵
3. 输出转置矩阵：
   - 外层循环：j从0到m-1（原矩阵的列，转置后的行）
   - 内层循环：i从0到n-1（原矩阵的行，转置后的列）
   - 输出a[i][j]

### 示例演示
```
原矩阵（2×3）：
a[0][0]=1  a[0][1]=2  a[0][2]=3
a[1][0]=4  a[1][1]=5  a[1][2]=6

转置输出（3×2）：
j=0: a[0][0]=1, a[1][0]=4  → 输出：1 4
j=1: a[0][1]=2, a[1][1]=5  → 输出：2 5
j=2: a[0][2]=3, a[1][2]=6  → 输出：3 6
```

### 关键点
- **索引交换**：原矩阵a[i][j]对应转置后的位置[j][i]
- **循环顺序**：先遍历列j，再遍历行i
- **输出格式**：注意空格和换行的位置

## 测试用例

### 输入
```
2 3
1 2 3
4 5 6
```

### 输出
```
1 4
2 5
3 6
```

### 验证
- 原矩阵2行3列，转置后3行2列 ✓
- a[0][0]=1 → 转置后第0行第0列 ✓
- a[1][0]=4 → 转置后第0行第1列 ✓

## 常见错误

1. **循环顺序错误**：应该先j后i，而不是先i后j
2. **输出格式错误**：每行最后一个数字后不应该有空格
3. **索引混淆**：转置时索引要交换
