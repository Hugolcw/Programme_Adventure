# 答案07：递归求幂

## 完整代码

见 `答案07_函数题_递归求幂.cpp`

## 解题思路

### 核心算法
使用递归计算幂：x^n = x * x^(n-1)，递归终止条件是n=0时返回1。

### 算法步骤
1. **终止条件**：n == 0，返回1.0
2. **正指数**：n > 0，返回 x * power(x, n-1)
3. **负指数**：n < 0，返回 1.0 / power(x, -n)

### 示例演示
```
power(2, 3):
  n=3>0: return 2 * power(2, 2)
    n=2>0: return 2 * power(2, 1)
      n=1>0: return 2 * power(2, 0)
        n=0: return 1.0
      返回: 2 * 1 = 2
    返回: 2 * 2 = 4
  返回: 2 * 4 = 8

power(2, -2):
  n=-2<0: return 1.0 / power(2, 2)
    power(2, 2) = 4
  返回: 1.0 / 4 = 0.25
```

### 关键点
- **递归终止**：n=0是终止条件
- **负指数处理**：转换为正指数后取倒数
- **数据类型**：使用double保证精度

## 测试用例

### 输入
```
3
2 3
2 -2
3 0
```

### 输出
```
8.000
0.250
1.000
```

### 验证
- 2^3 = 8 ✓
- 2^(-2) = 1/4 = 0.25 ✓
- 3^0 = 1 ✓

## 常见错误

1. **终止条件错误**：忘记处理n=0的情况
2. **负指数处理错误**：应该先转换为正指数再递归
3. **数据类型错误**：使用int可能导致精度丢失
4. **递归深度**：注意n的范围，避免栈溢出
