# 答案02：完美数判断

## 完整代码

见 `答案02_循环逻辑_完美数判断.cpp`

## 解题思路

### 核心算法
遍历1到n-1的所有数，找出能整除n的数（真因子），求和后判断是否等于n。

### 算法步骤
1. 初始化 `sum = 0`
2. 遍历 `j` 从1到n-1
3. 如果 `n % j == 0`，说明j是n的因子，累加到sum
4. 遍历结束后，判断 `sum == n`

### 示例演示
```
n = 6
j=1: 6%1==0, sum=1
j=2: 6%2==0, sum=3
j=3: 6%3==0, sum=6
j=4: 6%4!=0, 跳过
j=5: 6%5!=0, 跳过
结果：sum=6 == 6，输出"yes"
```

### 优化思路
可以优化到遍历到sqrt(n)，但需要处理完全平方数的情况。本题数据范围较小，直接遍历即可。

## 测试用例

### 输入
```
3
6
28
10
```

### 输出
```
yes
yes
no
```

### 验证
- 6的真因子：1, 2, 3，和=6 ✓
- 28的真因子：1, 2, 4, 7, 14，和=28 ✓
- 10的真因子：1, 2, 5，和=8 ≠ 10 ✗

## 常见错误

1. **包含自身**：循环应该是`j < n`而不是`j <= n`
2. **初始化错误**：sum应该初始化为0
3. **输出格式错误**：注意输出"yes"/"no"后要换行
