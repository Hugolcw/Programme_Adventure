# 答案04：数组元素查找

## 完整代码

见 `答案04_数组操作_数组元素查找.cpp`

## 解题思路

### 核心算法
线性查找（顺序查找）：从数组第一个元素开始，逐个比较，找到第一个匹配的元素。

### 算法步骤
1. 读取数组大小n和目标值target
2. 读取n个数组元素
3. 初始化 `pos = -1`（表示未找到）
4. 遍历数组，如果找到匹配元素：
   - 设置 `pos = i + 1`（位置从1开始）
   - 使用break提前退出循环
5. 输出pos

### 关键点
- **提前退出**：找到第一个匹配元素后立即break，提高效率
- **位置计数**：数组索引从0开始，但题目要求位置从1开始，所以输出时+1
- **未找到处理**：初始化为-1，如果没找到就输出-1

## 测试用例

### 测试用例1
**输入**：
```
5 3
1 2 3 4 5
```
**输出**：
```
3
```
**说明**：3在数组的第3个位置（索引为2）

### 测试用例2
**输入**：
```
5 6
1 2 3 4 5
```
**输出**：
```
-1
```
**说明**：6不在数组中

## 常见错误

1. **位置计数错误**：忘记+1，导致输出索引而不是位置
2. **未找到处理**：没有初始化pos为-1
3. **提前退出**：找到后没有break，可能找到后面的匹配元素
