# 题目12：数组最大值交换

## 题目类型
指针与引用题

## 难度
中等

## 知识点
指针、数组、函数参数传递

## 题目描述

编写一个函数，使用指针找到数组中的最大值，并将其与数组的第一个元素交换。

函数声明：`void swapMax(int *arr, int n)`
- arr：指向数组的指针
- n：数组元素个数

## 输入格式

第一行输入一个整数t（1 <= t <= 100），表示测试用例的数量。
对于每个测试用例：
- 第一行输入一个整数n（1 <= n <= 100），表示数组元素个数
- 第二行输入n个整数，表示数组元素

## 输出格式

对于每个测试用例，输出交换后的数组，元素之间用空格分隔。

## 样例输入

```
2
5
3 1 4 1 5
4
10 20 30 40
```

## 样例输出

```
5 1 4 1 3
40 20 30 10
```

## 提示

- 使用指针遍历数组
- 找到最大值的索引
- 交换第一个元素和最大值元素
- 注意指针的解引用操作

---

## 要求

1. 使用指针作为函数参数
2. 自己实现查找和交换逻辑
3. 注意数组边界
