# 题目10：学生成绩管理系统

## 题目类型
综合应用题

## 难度
较难

## 知识点
数组、字符串、循环、条件判断、统计计算

## 题目描述

设计一个简单的学生成绩管理系统：
1. 输入n个学生的姓名和3门课程的成绩
2. 计算每个学生的总分和平均分
3. 找出总分最高的学生
4. 统计各分数段的人数（90-100为A，80-89为B，70-79为C，60-69为D，0-59为F）
5. 输出所有学生的信息，按总分降序排列

## 输入格式

第一行输入一个整数n（1 <= n <= 100），表示学生数量。
接下来n行，每行格式：姓名 成绩1 成绩2 成绩3
- 姓名是字符串（不超过20个字符，不包含空格）
- 成绩是整数（0-100之间）

## 输出格式

第一行：输出总分最高的学生信息，格式："最高分: [姓名] [总分] [平均分]"
第二行：输出各分数段人数，格式："A: [人数] B: [人数] C: [人数] D: [人数] F: [人数]"
接下来n行：按总分降序输出所有学生信息，格式："[姓名] [总分] [平均分]"
（平均分保留1位小数）

## 样例输入

```
3
张三 85 90 88
李四 92 88 95
王五 78 82 80
```

## 样例输出

```
最高分: 李四 275 91.7
A: 1 B: 2 C: 0 D: 0 F: 0
李四 275 91.7
张三 263 87.7
王五 240 80.0
```

## 提示

- 使用结构体或并行数组存储学生信息
- 自己实现排序算法（冒泡排序或选择排序）
- 注意平均分的计算和输出格式

---

## 要求

1. 使用普通数组存储数据（不使用STL容器）
2. 自己实现排序算法
3. 注意输出格式的精确性
4. 代码结构清晰，逻辑正确
