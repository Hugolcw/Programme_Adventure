# 题目04：数组元素查找

## 题目类型
数组操作题

## 难度
中等

## 知识点
一维数组、循环、查找算法

## 题目描述

给定一个整数数组和一个目标值，找出目标值在数组中第一次出现的位置（从1开始计数）。如果不存在，输出-1。

## 输入格式

第一行输入两个整数n和target（1 <= n <= 1000，-1000 <= target <= 1000），n表示数组元素个数，target表示要查找的目标值。
第二行输入n个整数，表示数组元素（每个整数在-1000到1000之间）。

## 输出格式

输出目标值在数组中第一次出现的位置（从1开始），如果不存在输出-1。

## 样例输入

```
5 3
1 2 3 4 5
```

## 样例输出

```
3
```

## 样例输入2

```
5 6
1 2 3 4 5
```

## 样例输出2

```
-1
```

## 提示

- 使用循环遍历数组
- 找到第一个匹配的元素就返回位置
- 注意位置从1开始计数（不是从0开始）

---

## 要求

1. 使用普通数组存储数据
2. 自己实现查找算法（不使用STL的find）
3. 注意边界情况（数组为空、目标值不存在）
