# 题目03：数字序列

## 题目类型
循环逻辑题

## 难度
中等

## 知识点
循环、递推、条件判断

## 题目描述

有一个数字序列，规律如下：
- 第1项是1
- 第2项是2
- 从第3项开始，每一项等于前两项之和

即：1, 2, 3, 5, 8, 13, 21, ...

输入一个正整数n，输出这个序列的第n项。

## 输入格式

第一行输入一个整数t（1 <= t <= 100），表示测试用例的数量。
接下来t行，每行输入一个正整数n（1 <= n <= 40）。

## 输出格式

对于每个测试用例，输出序列的第n项。

## 样例输入

```
3
1
3
7
```

## 样例输出

```
1
3
21
```

## 提示

- 这是一个类似斐波那契数列的递推问题
- 第1项：1
- 第2项：2
- 第3项：1+2=3
- 第4项：2+3=5
- 第5项：3+5=8
- 可以使用循环递推计算

---

## 要求

1. 使用循环实现递推
2. 注意前两项的特殊处理
3. 注意数据范围，使用int足够
