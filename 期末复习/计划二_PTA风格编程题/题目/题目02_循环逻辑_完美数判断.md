# 题目02：完美数判断

## 题目类型
循环逻辑题

## 难度
中等

## 知识点
循环、条件判断、数学计算

## 题目描述

完美数是指一个数等于它的所有真因子（不包括自身）之和。

例如：6 = 1 + 2 + 3，所以6是完美数。

输入一个正整数n，判断它是否为完美数。

## 输入格式

第一行输入一个整数t（1 <= t <= 100），表示测试用例的数量。
接下来t行，每行输入一个正整数n（1 <= n <= 10000）。

## 输出格式

对于每个测试用例，如果n是完美数，输出"yes"，否则输出"no"。

## 样例输入

```
3
6
28
10
```

## 样例输出

```
yes
yes
no
```

## 提示

- 真因子：能整除n且小于n的正整数
- 6的真因子：1, 2, 3（不包括6本身）
- 28的真因子：1, 2, 4, 7, 14（不包括28本身）
- 需要遍历1到n-1，找出所有能整除n的数并求和

---

## 要求

1. 使用循环找出所有真因子
2. 计算真因子之和
3. 判断是否等于原数
