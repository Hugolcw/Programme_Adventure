# 题目01：数字反转

## 题目类型
基础语法题

## 难度
简单

## 知识点
循环、数字操作、条件判断

## 题目描述

输入一个正整数n，将其各位数字反转后输出。

例如：输入123，输出321；输入100，输出1（注意前导零的处理）。

## 输入格式

第一行输入一个整数t（1 <= t <= 100），表示测试用例的数量。
接下来t行，每行输入一个正整数n（1 <= n <= 10^9）。

## 输出格式

对于每个测试用例，输出反转后的数字（不包含前导零）。

## 样例输入

```
3
123
100
987654321
```

## 样例输出

```
321
1
123456789
```

## 提示

- 使用循环提取各位数字
- 注意处理前导零的情况
- 可以使用取模和除法操作

---

## 要求

1. 使用普通变量和循环实现
2. 注意边界情况（如100反转后是1，不是001）
3. 代码简洁，逻辑清晰
