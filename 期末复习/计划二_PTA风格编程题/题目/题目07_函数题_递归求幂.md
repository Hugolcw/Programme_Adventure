# 题目07：递归求幂

## 题目类型
函数题

## 难度
中等

## 知识点
递归、函数设计、数学计算

## 题目描述

编写一个递归函数，计算x的n次幂（x^n）。

注意：n可能是负数，需要处理负指数的情况。

## 输入格式

第一行输入一个整数t（1 <= t <= 100），表示测试用例的数量。
接下来t行，每行输入两个数x和n（x为浮点数，-100 <= x <= 100；n为整数，-10 <= n <= 10）。

## 输出格式

对于每个测试用例，输出x的n次幂，保留3位小数。

## 样例输入

```
3
2 3
2 -2
3 0
```

## 样例输出

```
8.000
0.250
1.000
```

## 提示

- 递归终止条件：n == 0时返回1
- 递归关系：x^n = x * x^(n-1)（n > 0）
- 负指数：x^(-n) = 1 / x^n
- 注意处理n=0的情况

---

## 要求

1. 实现递归函数power(double x, int n)
2. 处理正指数、负指数和零指数
3. 注意精度问题（使用double类型）
