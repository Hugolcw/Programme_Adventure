# 题目06：字符替换

## 题目类型
字符串处理题

## 难度
中等

## 知识点
字符串操作、循环、字符处理

## 题目描述

给定一个字符串s和两个字符oldChar、newChar，将字符串s中所有的oldChar替换为newChar后输出。

## 输入格式

第一行输入一个字符串s（长度不超过1000）。
第二行输入两个字符oldChar和newChar，用空格分隔。

## 输出格式

输出替换后的字符串。

## 样例输入

```
hello world
l x
```

## 样例输出

```
hexxo worxd
```

## 样例输入2

```
programming
g G
```

## 样例输入2

```
proGramminG
```

## 提示

- 使用string类存储字符串
- 遍历字符串的每个字符
- 如果字符等于oldChar，则替换为newChar
- 注意字符串的访问方式：s[i]

---

## 要求

1. 使用string类（基础操作）
2. 遍历字符串并替换字符
3. 注意输入格式（字符串可能包含空格，使用getline）
