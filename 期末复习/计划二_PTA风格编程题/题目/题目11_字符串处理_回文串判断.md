# 题目11：回文串判断

## 题目类型
字符串处理题

## 难度
中等

## 知识点
字符串操作、循环、双指针思想

## 题目描述

判断一个字符串是否为回文串。回文串是指正读和反读都相同的字符串。

例如："level"、"radar"是回文串，"hello"不是回文串。

注意：只考虑字母和数字，忽略大小写和空格。

## 输入格式

第一行输入一个整数t（1 <= t <= 100），表示测试用例的数量。
接下来t行，每行输入一个字符串s（长度不超过1000）。

## 输出格式

对于每个测试用例，如果s是回文串，输出"yes"，否则输出"no"。

## 样例输入

```
3
level
Hello
A man a plan a canal Panama
```

## 样例输出

```
yes
no
yes
```

## 提示

- 忽略大小写：将所有字符转换为小写（或大写）再比较
- 忽略空格：跳过空格字符
- 只考虑字母和数字：忽略其他字符
- 可以使用双指针从两端向中间比较

---

## 要求

1. 使用string类处理字符串
2. 注意字符的大小写转换（使用tolower或toupper）
3. 注意跳过空格和其他非字母数字字符
