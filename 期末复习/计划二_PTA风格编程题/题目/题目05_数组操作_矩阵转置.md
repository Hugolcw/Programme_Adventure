# 题目05：矩阵转置

## 题目类型
数组操作题

## 难度
中等

## 知识点
二维数组、嵌套循环、矩阵操作

## 题目描述

给定一个n×m的矩阵，将其转置后输出。转置是指将矩阵的行和列互换。

例如：
```
原矩阵：
1 2 3
4 5 6

转置后：
1 4
2 5
3 6
```

## 输入格式

第一行输入两个整数n和m（1 <= n, m <= 100），表示矩阵的行数和列数。
接下来n行，每行m个整数，表示矩阵元素。

## 输出格式

输出转置后的矩阵，共m行，每行n个整数，整数之间用空格分隔。

## 样例输入

```
2 3
1 2 3
4 5 6
```

## 样例输出

```
1 4
2 5
3 6
```

## 提示

- 原矩阵a[i][j]对应转置矩阵b[j][i]
- 转置后矩阵的行数等于原矩阵的列数
- 转置后矩阵的列数等于原矩阵的行数

---

## 要求

1. 使用二维数组存储矩阵
2. 注意输出格式（空格和换行）
3. 可以创建新矩阵存储转置结果，也可以直接输出
