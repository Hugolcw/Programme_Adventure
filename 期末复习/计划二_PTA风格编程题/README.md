# 计划二：PTA风格编程题

## 📋 题目说明

本计划包含10-12道PTA风格的编程题，严格按照教学进度要求，注重思维训练和实际应用。

## 🎯 题目特点

- **注重思维**：考查逻辑思维和算法设计能力
- **注重应用**：题目场景贴近实际生活
- **注重基础**：扎实的基础语法和数据结构应用
- **格式严格**：输入输出格式精确到空格和换行

## 📚 知识点覆盖

### 基础语法（20%）
- 循环、条件、数组、字符串基础

### 算法思维（40%）
- 递推、递归、查找、排序

### 数据结构（20%）
- 数组操作、字符串处理

### 面向对象（20%）
- 类设计、继承、多态

## ⚠️ 技术限制

### ✅ 可以使用
- 普通数组：`int a[100]`, `int a[100][100]`
- string类基础操作：输入输出、长度、比较、拼接、字符访问
- 指针和引用：作为函数参数
- 基础算法：自己实现排序、查找等

### ❌ 禁止使用
- STL容器：vector, map, list等
- STL算法：sort, find, count等
- 复杂STL特性：迭代器、Lambda表达式等

## 📁 文件结构

- `题目/` - 所有题目文件
- `答案/` - 所有答案文件（包含.cpp代码和解析）

## 💡 使用建议

1. 先独立完成题目，不要直接看答案
2. 注意输入输出格式的精确性
3. 完成后对照答案，理解解题思路
4. 对于错题，重点复习相关知识点
5. 建议按顺序完成，从简单到复杂
