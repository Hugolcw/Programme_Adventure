# 题目06：代码填空 - 继承与构造函数

## 题目类型
代码填空题

## 知识点
Lesson 5: OOP下 - 继承、构造函数初始化列表

## 题目描述

请补充完整以下代码，实现一个Rectangle类继承自Shape类。

```cpp
#include <iostream>
using namespace std;

class Shape {
protected:
    double x, y;  // 中心点坐标
public:
    Shape(double x, double y) : x(x), y(y) {
        cout << "Shape constructor: (" << x << ", " << y << ")" << endl;
    }
    virtual double getArea() = 0;  // 纯虚函数
};

class Rectangle : public Shape {
private:
    double width, height;
public:
    // 请在此处补充构造函数
    // 参数：x, y（中心点坐标），width（宽度），height（高度）
    // 要求：使用初始化列表调用基类构造函数，并初始化width和height
    
    // 请在此处补充getArea函数
    // 返回矩形的面积（width * height）
    
    void display() {
        cout << "Rectangle: center(" << x << ", " << y 
             << "), size(" << width << "x" << height 
             << "), area=" << getArea() << endl;
    }
};

int main() {
    Rectangle rect(0, 0, 5, 3);
    rect.display();
    return 0;
}
```

## 要求
1. 补充Rectangle类的构造函数，使用初始化列表
2. 实现getArea()函数，计算矩形面积
3. 确保程序能够正确编译和运行

---

## 提示
- 派生类构造函数需要使用初始化列表调用基类构造函数
- 格式：`Derived(...) : Base(...), member1(...), member2(...) { }`
- 纯虚函数必须在派生类中实现
