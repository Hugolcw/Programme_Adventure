# 题目13：编程题 - 综合应用

## 题目类型
编程题

## 知识点
综合应用：类设计、STL、字符串处理

## 题目描述

设计一个简单的图书管理系统，要求如下：

1. **定义 `Book` 类**：
   - 私有成员：
     - `title`（string）：书名
     - `author`（string）：作者
     - `price`（double）：价格
   - 公有成员：
     - 构造函数：`Book(string t, string a, double p)`
     - Getter函数：`getTitle()`, `getAuthor()`, `getPrice()`
     - 显示函数：`void display()`，输出格式：`[书名] by [作者], Price: [价格]`

2. **使用map存储图书**：
   - 键（key）：书名（string）
   - 值（value）：Book对象
   - 实现添加、查找、删除功能

3. **主函数功能**：
   - 添加至少3本图书到map中
   - 使用STL算法查找价格最高的图书
   - 删除一本图书
   - 输出所有剩余图书的信息

## 输入格式

第一行：操作数量n（n >= 3）
接下来n行，每行格式：
- `ADD 书名 作者 价格` - 添加图书
- `FIND 书名` - 查找图书（如果找到，输出图书信息）
- `DELETE 书名` - 删除图书
- `MAX` - 输出价格最高的图书信息

最后输出所有剩余图书的信息

## 输出格式

- 查找成功：输出图书信息
- 查找失败：输出 `Book not found`
- 删除成功：输出 `Deleted: [书名]`
- 价格最高：输出 `Most expensive: [图书信息]`
- 最后输出所有剩余图书，每行一本

## 样例输入

```
5
ADD 算法导论 托马斯 89.5
ADD C++程序设计 张三 65.0
ADD 数据结构 李四 78.0
FIND 算法导论
MAX
DELETE C++程序设计
```

## 样例输出

```
[算法导论] by 托马斯, Price: 89.50
Most expensive: [算法导论] by 托马斯, Price: 89.50
Deleted: C++程序设计
[算法导论] by 托马斯, Price: 89.50
[数据结构] by 李四, Price: 78.00
```

## 要求

1. 使用map存储图书（书名作为键）
2. 使用STL算法查找价格最高的图书
3. 注意输出格式（价格保留2位小数）
4. 处理查找失败的情况

---

## 提示

- map的键是唯一的，重复添加会覆盖
- 可以使用max_element配合Lambda表达式查找价格最高的图书
- 注意map的迭代器类型
