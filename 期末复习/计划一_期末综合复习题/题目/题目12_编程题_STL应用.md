# 题目12：编程题 - STL应用

## 题目类型
编程题

## 知识点
Lesson 8-9: STL容器和算法

## 题目描述

编写一个程序，实现以下功能：

1. **使用vector存储学生成绩**：
   - 输入n个学生的成绩（整数，0-100）
   - 存储到vector中

2. **统计功能**：
   - 计算平均分（保留2位小数）
   - 找出最高分和最低分
   - 统计及格人数（>=60分）

3. **排序功能**：
   - 使用STL的sort算法对成绩进行降序排序
   - 输出排序后的成绩

4. **查找功能**：
   - 输入一个成绩值
   - 使用STL的find算法查找该成绩
   - 如果找到，输出"Found"，否则输出"Not found"

## 输入格式

第一行：学生数量n（n > 0）
第二行：n个成绩，用空格分隔
第三行：要查找的成绩值

## 输出格式

第一行：`Average: [平均分]`
第二行：`Max: [最高分], Min: [最低分]`
第三行：`Pass: [及格人数]`
第四行：排序后的成绩（降序），用空格分隔
第五行：查找结果（"Found" 或 "Not found"）

## 样例输入

```
5
85 92 58 76 90
76
```

## 样例输出

```
Average: 80.20
Max: 92, Min: 58
Pass: 4
92 90 85 76 58
Found
```

## 要求

1. 必须使用STL容器（vector）和算法（sort、find、count_if等）
2. 使用范围for循环输出（C++11）
3. 注意输出格式的精确性

---

## 提示

- `sort(v.begin(), v.end(), greater<int>())` 实现降序排序
- `find(v.begin(), v.end(), value)` 查找元素
- `count_if` 可以统计满足条件的元素个数
