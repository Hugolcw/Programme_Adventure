# 题目05：代码填空 - 函数重载

## 题目类型
代码填空题

## 知识点
Lesson 2: 函数和作用域 - 函数重载

## 题目描述

请补充完整以下代码，实现函数重载：根据参数类型的不同，计算两个数的和。

```cpp
#include <iostream>
using namespace std;

// 请在此处填写函数声明和定义
// 需要实现三个版本的add函数：
// 1. 两个int参数，返回int
// 2. 两个double参数，返回double
// 3. 两个string参数，返回string（字符串拼接）

int main() {
    cout << add(3, 5) << endl;           // 应该输出: 8
    cout << add(3.5, 2.5) << endl;        // 应该输出: 6
    cout << add(string("Hello"), string("World")) << endl;  // 应该输出: HelloWorld
    
    return 0;
}
```

## 要求
1. 在代码中补充三个`add`函数的声明和定义
2. 确保函数重载正确，能够根据参数类型自动选择对应的函数
3. 字符串版本需要将两个字符串拼接后返回

---

## 提示
- 函数重载要求函数名相同，但参数列表不同（类型或数量）
- 字符串拼接可以使用 `+` 运算符
