# 题目10：代码改错 - 数组越界

## 题目类型
代码改错题

## 知识点
Lesson 3: 数组和指针 - 数组边界检查

## 题目描述

以下代码存在数组越界问题，请找出并修正所有错误。

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[5] = {1, 2, 3, 4, 5};
    
    // 错误1：数组越界访问
    for (int i = 0; i <= 5; i++) {  // 应该是 i < 5
        cout << arr[i] << " ";
    }
    cout << endl;
    
    // 错误2：使用负数索引
    cout << arr[-1] << endl;  // 未定义行为
    
    // 错误3：访问超出范围
    cout << arr[10] << endl;  // 数组越界
    
    return 0;
}
```

## 要求
1. 找出所有数组越界相关的错误
2. 修正错误，确保数组访问安全
3. 说明如何避免数组越界

---

## 提示
- 数组索引范围是 [0, size-1]
- 循环条件应该是 `i < size` 而不是 `i <= size`
- 可以使用 `vector` 的 `at()` 方法进行边界检查
