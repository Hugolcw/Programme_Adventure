# 题目11：编程题 - 学生管理系统

## 题目类型
编程题

## 知识点
Lesson 4-6: OOP - 类、继承、多态

## 题目描述

设计一个学生管理系统，要求如下：

1. **定义基类 `Person`**：
   - 保护成员：`name`（string类型）、`id`（int类型）
   - 公有成员：
     - 构造函数：`Person(string name, int id)`
     - 虚函数：`virtual void display() = 0`（纯虚函数）

2. **定义派生类 `Student`**：
   - 继承自 `Person`
   - 私有成员：`scores`（int数组，大小为3，存储3门课程成绩）
   - 公有成员：
     - 构造函数：`Student(string name, int id, int scores[])`
     - 实现 `display()` 函数：输出学生信息（姓名、学号、平均分，保留1位小数）

3. **定义派生类 `Teacher`**：
   - 继承自 `Person`
   - 私有成员：`salary`（double类型）
   - 公有成员：
     - 构造函数：`Teacher(string name, int id, double salary)`
     - 实现 `display()` 函数：输出教师信息（姓名、工号、工资，保留2位小数）

4. **主函数要求**：
   - 创建至少2个Student对象和1个Teacher对象
   - 使用基类指针数组存储这些对象
   - 使用多态调用 `display()` 函数输出所有信息

## 输入格式

第一行输入学生数量n（n >= 2）
接下来n行，每行输入：学生姓名 学号 成绩1 成绩2 成绩3
接下来1行，输入：教师姓名 工号 工资

## 输出格式

先输出所有学生信息，每行格式：`姓名 学号 平均分`
然后输出教师信息，格式：`姓名 工号 工资`

## 样例输入

```
2
张三 1001 85 90 88
李四 1002 92 88 95
王老师 2001 8000.5
```

## 样例输出

```
张三 1001 87.7
李四 1002 91.7
王老师 2001 8000.50
```

## 要求

1. 使用继承和多态实现
2. 基类析构函数应该声明为虚函数
3. 注意内存管理，避免内存泄漏
4. 输出格式要精确（小数位数）

---

## 提示

- 纯虚函数使Person成为抽象类，不能直接实例化
- 使用基类指针数组实现多态
- 记得释放动态分配的内存
