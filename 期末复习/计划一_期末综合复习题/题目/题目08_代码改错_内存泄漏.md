# 题目08：代码改错 - 内存泄漏

## 题目类型
代码改错题

## 知识点
Lesson 3: 数组和指针 - 动态内存分配

## 题目描述

以下代码存在内存泄漏问题，请找出并修正所有错误。

```cpp
#include <iostream>
using namespace std;

int* createArray(int size) {
    int* arr = new int[size];
    for (int i = 0; i < size; i++) {
        arr[i] = i * 2;
    }
    return arr;
}

int main() {
    int* ptr1 = createArray(5);
    int* ptr2 = createArray(10);
    
    for (int i = 0; i < 5; i++) {
        cout << ptr1[i] << " ";
    }
    cout << endl;
    
    for (int i = 0; i < 10; i++) {
        cout << ptr2[i] << " ";
    }
    cout << endl;
    
    delete ptr1;
    // 问题：缺少对ptr2的delete
    
    return 0;
}
```

## 要求
1. 找出代码中的所有错误
2. 修正错误，确保没有内存泄漏
3. 说明每个错误的原因

---

## 提示
- 使用 `new` 分配的内存必须用 `delete` 释放
- 使用 `new[]` 分配的内存必须用 `delete[]` 释放
