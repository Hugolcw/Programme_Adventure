# 题目02：代码阅读 - 多态与虚函数

## 题目类型
代码阅读题

## 知识点
Lesson 6: 多态 - 虚函数、动态绑定

## 题目描述

阅读以下代码，写出程序的输出结果。

```cpp
#include <iostream>
using namespace std;

class Animal {
public:
    virtual void speak() {
        cout << "Animal speaks" << endl;
    }
    void eat() {
        cout << "Animal eats" << endl;
    }
};

class Dog : public Animal {
public:
    void speak() override {
        cout << "Dog barks" << endl;
    }
    void eat() {
        cout << "Dog eats bones" << endl;
    }
};

int main() {
    Animal* ptr = new Dog();
    ptr->speak();
    ptr->eat();
    delete ptr;
    return 0;
}
```

## 要求
请写出程序的完整输出结果（包括所有换行）。

---

## 提示
注意虚函数和非虚函数的区别。
