# 答案02：代码阅读 - 多态与虚函数

## 正确答案

```
Dog barks
Animal eats
```

## 解析

### 虚函数 vs 非虚函数

1. **`speak()` 是虚函数**：
   - 使用 `virtual` 关键字声明
   - 通过基类指针调用时，会根据**实际对象类型**（Dog）调用派生类的版本
   - 输出：`Dog barks`

2. **`eat()` 不是虚函数**：
   - 没有 `virtual` 关键字
   - 通过基类指针调用时，根据**指针类型**（Animal*）调用基类的版本
   - 输出：`Animal eats`

### 关键知识点
- **虚函数实现多态**：基类指针指向派生类对象时，调用虚函数会执行派生类的版本
- **非虚函数不实现多态**：根据指针类型决定调用哪个函数
- **多态的条件**：
  1. 基类指针或引用
  2. 虚函数
  3. 派生类重写虚函数

### 常见错误
- 忘记在基类函数前加 `virtual` 关键字，导致多态失效
- 误以为所有函数都能实现多态
