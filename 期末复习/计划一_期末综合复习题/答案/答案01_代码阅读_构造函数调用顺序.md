# 答案01：代码阅读 - 构造函数调用顺序

## 正确答案

```
Creating object...
Base constructor
Derived constructor
Object created
Derived destructor
Base destructor
```

## 解析

### 构造函数调用顺序
1. 创建派生类对象时，**先调用基类构造函数**，再调用派生类构造函数
2. 因此输出顺序为：
   - `Base constructor`
   - `Derived constructor`

### 析构函数调用顺序
1. 对象销毁时，**先调用派生类析构函数**，再调用基类析构函数
2. 因此输出顺序为：
   - `Derived destructor`
   - `Base destructor`

### 关键知识点
- **继承中的构造/析构顺序**：
  - 构造：基类 → 派生类
  - 析构：派生类 → 基类
- 这是C++面向对象编程的重要特性，确保基类部分先初始化，派生类部分后销毁。
