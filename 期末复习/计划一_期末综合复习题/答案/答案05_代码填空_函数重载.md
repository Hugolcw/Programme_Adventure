# 答案05：代码填空 - 函数重载

## 正确答案

```cpp
#include <iostream>
#include <string>
using namespace std;

// 函数声明
int add(int a, int b);
double add(double a, double b);
string add(string a, string b);

// 函数定义
int add(int a, int b) {
    return a + b;
}

double add(double a, double b) {
    return a + b;
}

string add(string a, string b) {
    return a + b;
}

int main() {
    cout << add(3, 5) << endl;           // 输出: 8
    cout << add(3.5, 2.5) << endl;        // 输出: 6
    cout << add(string("Hello"), string("World")) << endl;  // 输出: HelloWorld
    
    return 0;
}
```

## 解析

### 函数重载的关键点

1. **函数名相同**：三个函数都叫 `add`
2. **参数列表不同**：
   - `add(int, int)` - 两个int参数
   - `add(double, double)` - 两个double参数
   - `add(string, string)` - 两个string参数
3. **编译器自动选择**：根据传入参数的类型，编译器会自动选择匹配的函数

### 注意事项

1. **需要包含 `<string>` 头文件**：使用string类型需要包含此头文件
2. **字符串拼接**：使用 `+` 运算符可以直接拼接两个string对象
3. **函数声明**：可以先声明，后定义；也可以直接定义

### 常见错误

- 忘记包含 `<string>` 头文件
- 参数类型不匹配（如传入 `"Hello"` 而不是 `string("Hello")`）
- 返回值类型与函数声明不一致

### 扩展知识

函数重载还可以通过**参数数量**不同来实现，例如：
```cpp
int add(int a, int b);
int add(int a, int b, int c);  // 参数数量不同
```
