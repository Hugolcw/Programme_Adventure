# 计划一：期末综合复习题

## 📋 题目说明

本计划包含10-13道综合复习题，全面覆盖C++课程的10个章节知识点。

## 📚 题目类型分布

- **代码阅读题**（3-4题）：理解代码执行流程，写出输出结果
- **代码填空题**（2-3题）：补充关键代码，完成功能
- **代码改错题**（2-3题）：找出并修正代码中的错误
- **编程题**（2-3题）：根据需求编写完整程序

## 🎯 知识点覆盖

### 基础篇（Lesson 1-3）- 20%
- 数据类型、运算符、控制流
- 函数定义、参数传递、作用域
- 数组操作、指针使用、动态内存

### 面向对象篇（Lesson 4-6）- 40%
- 类与对象、构造函数、访问控制
- 继承、构造函数初始化列表
- 虚函数、多态、动态绑定

### 高级特性篇（Lesson 7-10）- 40%
- 智能指针（unique_ptr、shared_ptr）
- STL容器（vector、map、string）
- STL算法（sort、find、transform）
- 移动语义（选考）

## 💡 使用建议

1. 先独立完成题目，不要直接看答案
2. 完成后对照答案，理解解题思路
3. 对于错题，重点复习相关知识点
4. 可以多次练习，加深理解
5. 建议按顺序完成，从简单到复杂
