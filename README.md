# 第一次学习：C++基础语法与数据类型

## 学习内容
- 变量和常量声明
- 基本数据类型  
- 运算符使用
- 控制流语句（if-else, for循环, switch-case）
- 输入输出操作

## 练习文件

### 练习1: 圆面积计算
- 文件：`circle_calculator.cpp`
- 功能：计算半径为5的圆的面积和周长

### 练习2: 偶数判断
- 文件：`even_checker.cpp`
- 功能：判断输入的整数是奇数还是偶数

### 练习3: 3的倍数
- 文件：`multiples_of_three.cpp`
- 功能：输出1-100之间所有3的倍数

### 练习4: 四则运算计算器
- 文件：`calculator.cpp`
- 功能：简单的四则运算计算器
